# nestjs-products

## Description

This server application is a product store built in Nest.js, it's a product CRUD, it also allows uploading images. It's client app is called [reactjs-products](https://github.com/eavelasquez/reactjs-products/)

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
